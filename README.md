An interactive Business Intelligence dashboard developed using **Power BI** to analyze global sales, profitability, discount impact, and regional performance across products, countries, and customer segments.

---

## Project Overview

This project analyzes **20,000+ structured financial sales records** to uncover revenue trends, profitability drivers, discount impact, and underperforming segments.

The dashboard is divided into three analytical report pages:

1. Sales Performance Overview  
2. Regional Sales & Profitability  
3. Discount & Profitability Insights  

The solution enables stakeholders to make data-driven strategic decisions through dynamic KPIs and drill-down analytics.

---

## Business Problem

Organizations often struggle to:

- Identify high-profit products  
- Monitor regional sales performance  
- Track time-based revenue growth (YTD, monthly trends)  
- Detect underperforming customer segments  
- Understand the impact of discounts on profitability  

This dashboard solves these challenges through interactive KPI tracking and filtering.

---

## Dataset

- Records: 20,000+  
- Fields: Product, Country, Segment, Sales, Profit, Discount, Date, Units Sold  
- Data Type: Structured financial sales dataset  

---

## Tools & Technologies

- Power BI  
- Power Query (Data Transformation)  
- DAX (Data Analysis Expressions)  
- Time Intelligence Functions  
- Data Modeling  

---

## Implementation Steps

1. Cleaned and transformed raw dataset using Power Query  
2. Built relational data model  
3. Created advanced DAX measures:
   - Profit Margin  
   - YTD Sales  
   - Segment Performance  
   - Regional Contribution  
   - Discount Analysis  
4. Designed interactive dashboards with:
   - KPI Cards  
   - Drill-through pages  
   - Slicers  
   - Map visualizations  
   - Time-based trend analysis
